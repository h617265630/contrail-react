## Design Context

### Users
- **Mixed audience**: Students and professionals learning for career and personal development
- **Context**: Self-directed learning - users choose when/what to learn
- **Goal**: Turn scattered resources into structured, actionable knowledge

### Brand Personality
- **Energetic & motivating** — dynamic, action-inspiring, but not aggressive
- **Personal** — feels like a trusted tool, not an enterprise platform
- **Clear** — information-dense but never cluttered

### Aesthetic Direction
- **Editorial warmth**: Stone warm neutrals paired with amber accents create an approachable, focused feel
- **Serif headlines**: Georgia/serif for display text adds sophistication and editorial credibility
- **Clean utility**: Tailwind CSS with custom theme tokens, minimal decoration, function-first
- **Card-based layouts**: Information organized in clear card containers with subtle borders

### Design Principles
1. **Warm minimalism** — Remove, don't add. Every element earns its place.
2. **Editorial hierarchy** — Serif headlines create visual interest; sans-serif body maintains readability
3. **Accent with purpose** — Amber for primary actions, stone for structure, no decorative color noise
4. **Consistent rhythm** — Fixed spacing scale (2/4/6/8/12px), rounded-sm/rounded-md corners
5. **Motion with meaning** — Transitions serve function (state feedback, spatial orientation), never decoration
